# Native TDLib Bridge (Stub)

These files show how to register a Flutter MethodChannel (`spnetgram/tdlib`) for TDLib calls.

Once Flutter platform folders exist:
- Copy `android/TdlibBridge.kt` and update your `MainActivity.kt`.
- Copy `ios/TdlibBridge.swift` and hook in `AppDelegate.swift`.
- Replace TODOs with TDLib client initialization and updates.
